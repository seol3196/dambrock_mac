# 담벼락 작업 내용

## 현재 상태

- 경로: `C:\dev\dambrock`
- 스택: React + Vite + Firebase Auth + Firestore + Tailwind CSS
- 로컬 서버: `http://127.0.0.1:5173`
- Firebase 프로젝트: `dambrock`

## 구현 완료

- 역할별 라우팅: `/`, `/admin`, `/teacher`, `/student`, `/wall/:wallId`
- 관리자: 교사 계정 생성
- 교사: 학생 일괄 생성, 담벼락 생성/관리
- 학생: 담벼락 목록 진입
- 담벼락: 글 작성, 댓글, 좋아요, 삭제
- 학생 본인 글 수정 가능
- 링크 자동 인식 및 클릭 가능
- 교사 담벼락 페이지 공유 기능 추가
  - QR 코드
  - 링크 복사
  - 로그인 필요 토글
- 교사 담벼락 관리 목록도 “로그인 필요” 토글로 정리
- 교사가 학생 포스트잇 위치를 열 기준으로 조정 가능
- 학생/담벼락 화면 레이아웃 확장, 포스트잇 폭 축소
- 포스트잇 회전 제거, 본문 글씨 가독성 개선

## Firebase 상태

- `admin@damvyeorak.local` / `admin1234` 로그인 확인
- `scripts/create-admin.mjs`로 admin 문서 부트스트랩 완료
- `firestore.rules`는 로컬 수정 완료
- 남은 외부 작업: Firebase CLI 로그인 후 rules 배포

```bash
npx firebase-tools login
firebase deploy --only firestore:rules
```
