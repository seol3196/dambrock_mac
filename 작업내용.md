# 담벼락 작업 내용

## 현재 상태

- 경로: `C:\dev\dambrock`
- 스택: React + Vite + Firebase Auth + Firestore + Tailwind CSS
- Firebase 프로젝트: `dambrock`
- Firestore rules 배포 완료

## 구현 완료

- 역할별 경로: `/`, `/admin`, `/teacher`, `/student`, `/wall/:wallId`
- 관리자: 교사 계정 생성
- 교사:
  - 학생 한 명 생성
  - 학생 일괄 생성
  - 학생 이름 수정
  - 개별 비밀번호 변경
  - 선택 학생 비밀번호 일괄 변경
  - 개별 `123456` 초기화
  - 학생 삭제
  - 담벼락 생성 / 목록 / 공유 / 설정
- 학생:
  - 로그인 후 담벼락 진입
  - 게시글 작성
  - 본인 글 수정 / 삭제
- 담벼락:
  - 코르크 배경 위에 포스트잇 직접 배치
  - 컬럼 수 1~5 설정, 기본 4
  - 배경 색상 변경
  - 로그인 필요 토글
  - 댓글 / 좋아요 토글
  - 링크 복사 / QR 공유
  - 교사 포스트잇 드래그 이동

## 최근 수정

- 로그인 필요한 담벼락 링크 진입 후 로그인하면 원래 담벼락으로 복귀하도록 수정
- 학생 링크 진입 시 인증 로딩 순서 때문에 담벼락이 비던 문제 수정
- 교사 페이지 문구/구조 정리
- Netlify 배포용 `netlify.toml` 추가

## Firebase 메모

- 관리자 로그인: `admin` / `admin1234`
- 학생 삭제 / 비밀번호 변경은 Firebase Functions 경로 사용
- Functions는 필요 시 별도 배포 필요

## 배포 메모

- 빌드: `npm run build`
- Netlify:
  - 저장소 연결 후 `main` 배포
  - 환경변수에 `VITE_FIREBASE_*` 값 등록
  - `netlify.toml`로 SPA 라우팅 처리
