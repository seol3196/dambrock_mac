1. 완료. 교사만 포스트잇을 드래그해서 1~4열로 이동 가능하게 변경.
2. 완료. 포스트잇 상단 아이콘 배치를 정리해 오른쪽 여백 낭비 감소.
3. 완료. WALL INFO를 교사용 설정 패널로 전환하고, 우상단 `설정` 버튼으로만 열리게 변경. 제목/설명/로그인 필요/댓글/좋아요/배경색 수정 가능.
4. 완료. 학생 화면에서는 설정 패널과 컬럼 드롭존을 숨기고 포스트잇 영역을 더 넓게 사용하도록 조정.
5. 완료. 교사가 학생 이름을 수정할 수 있게 했고, 학생 목록은 이름 기준으로 정렬되도록 변경.
6. 완료. 학생 생성 시작 번호 기본값을 `01`로 변경하고 자리수를 유지해 ID 생성.
7. 완료. 교사 모드 사이드바의 `담벼락 관리` 명칭을 `담벼락`으로 변경.
8. 완료. 담벼락 목록 카드에서는 설정 토글을 제거하고, 변경은 각 담벼락 내부 설정 패널에서만 하도록 정리.
9. 완료. 담벼락 목록에 각 담벼락 생성 시간 표시 추가.
